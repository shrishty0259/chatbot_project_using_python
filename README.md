# chatbot_project_using_python
# Python Chatbot

This is a simple chatbot built using Python, NLP, and TensorFlow.

## Features
- Responds to greetings
- Answers basic questions
- Uses machine learning model

## 🛠️ Technologies Used
- Python
- NLTK
- TensorFlow

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Train the model:
python train_only.py

3. Run chatbot:
python chatbot.py

##  Files
- chatbot.py → main chatbot
- train_only.py → training model
- intents.json → training data
